<h2 id='2'>Подключение</h2><a href="#0"></a>
<ul>
	<li>Колонки имеют входы: MIC (микрофонные) и LINE (линейные).</li><br>
	<li>Микрофонные входы предназначены для подключения микрофонов.</li><br>
	<li>Линейные входы предназначены для подключения всех остальных источников звука. Например линейный выход из одной колонки подключается на линейный вход другой колонки. </li>
</ul>
